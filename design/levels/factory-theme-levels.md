# 工厂主题区 — 关卡设计文档

## Overview

主题区：星港分拣工厂 (Toy Port Sorting Factory)
风格：工业仓储，明亮安全，橙黄色暖调 + 金属质感

**目标**：在教学关之后，用 6 个关卡验证玩家已掌握：
1. 箱子状态管理（功能面朝向）
2. 地形机制对箱子状态的影响
3. 多种地形机制的组合使用

---

## Level 6: 斜坡导引

### Metadata

- **编号**: Level 6
- **区域**: 工厂入口卸货区 D-01
- **类型**: Terrain Tutorial
- **目标**: 玩家首次接触坡道，理解坡道会改变箱子朝向
- **状态**: Concept

### 核心教学

- 坡道不移动箱子，只改变箱子朝向
- 箱子滚入坡道时，顶面按规则旋转
- 箱子需要正确朝向才能完成后续任务

### 关卡布局

```
##########
#S.......#
#........#
#...B....#
#...R....#
#........#
#.......E#
##########

S = 玩家起点 (2,1)
B = 箱子 (3,3)
R = 坡道，方向向上 (3,4)
E = 出口 (8,6)
```

### 关键设计

- 箱子初始为 NORMAL_A 朝上
- 出口处有一个 NORMAL 按钮，需要箱子以 NORMAL 面压住才能开门
- 玩家必须先把箱子滚到坡道上（变成正确面），再推往出口
- 不需要坡道也能过关（直接滚动调整），但更慢——坡道是"正确解"

### Critical Path

1. 玩家推动箱子，发现直接推向出口无法触发按钮（顶面不对）
2. 玩家注意到地图中间的坡道
3. 玩家把箱子推上传送带
4. 箱子经过坡道，朝向改变（变成 NORMAL）
5. 玩家把箱子从坡道推往出口
6. NORMAL 面压住按钮，门开，玩家过关

### 视觉引导

- 坡道用黄色箭头朝上标记，提示"这里是上行方向"
- 出口按钮旁发光提示"需要正确顶面"
- 教学提示：`坡道会改变箱子顶面。按方向滚动进入，顶面会旋转。`

## Level 7: 传送分拣线

### Metadata

- **编号**: Level 7
- **区域**: 工厂分拣轨道 D-02
- **类型**: Terrain Tutorial
- **目标**: 玩家首次接触传送带，理解传送带自动传输箱子
- **状态**: Concept

### 核心教学

- 传送带会自动推动箱子，无需玩家操作
- 玩家需要先规划箱子路线，再把箱子放到传送带上
- 传送带推动可以被墙壁阻挡

### 关卡布局

```
############
#S.........#
#..........#
#..B..C..C.#  C = 传送带，方向向右
#..........#
#.........E#
#..........#
############

S = 玩家起点 (1,1)
B = 箱子 (2,2)
C1 = 传送带 (5,2)
C2 = 传送带 (7,2)
E = 出口 (11,5)
```

### 关键设计

- 出口旁有 ENERGY 按钮，需要箱子以 ENERGY 面压住
- 箱子初始是 NORMAL_A
- 玩家需要在箱子滚到出口前调整好朝向
- 箱子经过两次传送带后到达出口区域

### Critical Path

1. 玩家把箱子推向传送带
2. 箱子被传送带自动向右传输（每格 0.8 秒）
3. 箱子经过两个传送带，到达出口附近
4. 玩家在箱子被传送带推过站之前，推动箱子一格（垂直方向）使其离开传送带
5. 玩家推动箱子到按钮处，发现顶面不对
6. 玩家重新滚箱子调整顶面，再送上传送带
7. ENERGY 面压住按钮，过关

### 视觉引导

- 传送带用动态箭头纹理，显示传输方向
- 教学提示：`传送带会自己移动箱子。把箱子放上去，它就会自己走。`

## Level 8: 旋转质检台

### Metadata

- **编号**: Level 8
- **区域**: 工厂质检站 D-03
- **类型**: Terrain Tutorial
- **目标**: 玩家首次接触旋转平台，理解顺时针 90° 旋转
- **状态**: Concept

### 核心教学

- 旋转平台会在箱子滚入时顺时针旋转箱子朝向 90°
- 可以多次旋转（每次 +90°）
- 旋转平台不移动箱子，只改变朝向

### 关卡布局

```
##########
#S..B....#
#........#
#....RP..#
#....E...#
##########

S = 玩家起点 (1,1)
B = 箱子 (3,1)
RP = 旋转平台 (5,3)
E = 出口 (5,4)
```

### 关键设计

- 出口旁有 IMPACT 按钮，需要箱子以 IMPACT 面压住
- 箱子初始是 NORMAL_A
- 旋转平台顺时针旋转箱子：NORMAL_A → 需要两次旋转才能变成 IMPACT
- 玩家必须先算出需要几次旋转

### Critical Path

1. 玩家把箱子推向旋转平台
2. 箱子经过旋转平台，顶面顺时针旋转 90°
3. 玩家观察顶面，发现不是 IMPACT
4. 玩家再次把箱子推入旋转平台（再次旋转 90°）
5. 现在是 IMPACT 面，压住按钮，过关

### 视觉引导

- 旋转平台有缓慢旋转动画（5 RPM），踩上去后加速（20 RPM）
- 教学提示：`旋转台每次踩上去，箱子会顺时针转 90°。`

## Level 9: 坡道+传送带组合

### Metadata

- **编号**: Level 9
- **区域**: 工厂转运通道 D-04
- **类型**: Combined Terrain
- **目标**: 玩家同时使用坡道和传送带，理解两者的协作逻辑
- **状态**: Concept

### 核心教学

- 坡道改变箱子朝向，传送带负责传输
- 两者组合：先调整朝向，再传输到目标区域
- 需要规划顺序：谁先谁后

### 关卡布局

```
##############
#S...........#
#............#
#..B.........#
#..R....C..C.E#
#............#
##############

S = 玩家起点 (1,1)
B = 箱子 (2,3)
R = 坡道，方向向上 (2,4)
C1 = 传送带 (6,4)
C2 = 传送带 (8,4)
E = 出口 (11,4)
```

### 关键设计

- 出口旁有 NORMAL 按钮
- 箱子初始是 IMPACT_A（需要变成 NORMAL）
- 坡道可以把 IMPACT 变成 NORMAL
- 传送带把箱子从坡道出口送到按钮旁

### Critical Path

1. 玩家把箱子推上坡道，朝向从 IMPACT 变成 NORMAL
2. 箱子从坡道落入传送带区域
3. 传送带把箱子向右传输
4. 箱子到达按钮旁，NORMAL 面压住按钮
5. 门开，玩家过关

### 视觉引导

- 坡道和传送带之间有明确的视觉连接（引导线）
- 教学提示：`坡道换方向，传送带远距离传输。配合使用效率最高。`

## Level 10: 三机制联动

### Metadata

- **编号**: Level 10
- **区域**: 工厂中央调度室 D-05
- **类型**: Combined Terrain (All Three)
- **目标**: 玩家同时使用全部三种地形机制
- **状态**: Concept

### 核心教学

- 三种机制按正确顺序组合使用
- 旋转平台用于精细调整（+90° 微调）
- 坡道用于大方向转换
- 传送带用于远距离传输

### 关卡布局

```
################
#S.............#
#..............#
#..B...........#
#..R.......RP.E#
#.........C....#
#..............#
################

S = 玩家起点 (1,1)
B = 箱子 (2,3)
R = 坡道，方向向上 (2,4)
C = 传送带，方向向右 (9,5)
RP = 旋转平台 (11,4)
E = 出口 (11,4)
```

### 关键设计

- 出口有 IMPACT 按钮
- 箱子初始需要经过：坡道（改变大方向）→ 传送带（传输）→ 旋转平台（+90° 微调）才能变成 IMPACT
- 路线不是直线，需要玩家规划

### Critical Path

1. 玩家把箱子推上坡道，箱子朝向改变
2. 箱子落入传送带，被向右传输
3. 箱子到达旋转平台，踩上后顺时针旋转 90°
4. 如果顶面变成 IMPACT，压住按钮过关
5. 如果不是，再推回去旋转一次

### 视觉引导

- 三种地形用不同颜色区分（黄=坡道，青=传送带，红=旋转）
- 教学提示：`三种地形一起用：坡道大转向，传送带传输，旋转台微调。`

## Level 11: 综合排序关

### Metadata

- **编号**: Level 11
- **区域**: 工厂发货终端 E-01
- **类型**: Final Puzzle
- **目标**: 在单一箱子上依次完成坡道→旋转→传送带的完整流程
- **状态**: Concept

### 核心教学

- 终极关卡：玩家需要精确控制箱子依次经历所有三种地形
- 箱子在三种地形之间来回多次，调整到精确朝向
- 考验玩家对"滚动方向=朝向变化"的理解深度

### 关卡布局

```
##################
#S...............#
#................#
#..B............#
#..R.........C..#
#..........RP...#
#...............#
#..............E#
##################

S = 玩家起点 (1,1)
B = 箱子 (2,3)
R = 坡道，方向向上 (2,4)
C = 传送带，方向向上 (10,4)
RP = 旋转平台 (8,5)
E = 出口 (14,7)
```

### 关键设计

- 出口有 ENERGY 按钮
- 箱子初始朝向需要经过复杂路径才能变成 ENERGY
- 需要玩家多次利用三种地形，反复调整箱子朝向
- 给玩家足够的试错空间

### Critical Path

1. 玩家把箱子推上坡道，第一次改变朝向
2. 箱子被传送带向上传输
3. 箱子到达旋转平台，旋转 90°
4. 玩家观察顶面，如果不满意，把箱子推回传送带，再传一次，再旋转
5. 直到顶面变成 ENERGY，压住按钮过关

### 教学总结

- 通过 Level 11 后，玩家应该完全理解三种地形机制及其组合
- 这是 Vertical Slice 的最终验证关卡
