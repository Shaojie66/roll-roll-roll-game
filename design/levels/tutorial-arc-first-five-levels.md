# 前 5 关教学设计

## Overview

这份文档定义《玩具星港：滚滚滚》的前 5 关教学弧线。目标不是做难关，而是在 20-30 分钟内把玩家从“看不懂箱子”带到“会主动规划同一箱子的多重用途”。

整体原则：
- 每关只引入 1 个新概念，最多叠加 1 个旧概念
- 每关都能在 3 分钟内首次通关
- 每关失败后都能在 5 秒内重新开始
- 第 5 关必须让玩家明确体验“一箱多用”

## Arc Summary

| 关卡 | 名称 | 新教学点 | 预计时长 | 难度 |
| ---- | ---- | ---- | ---- | ---- |
| 1 | 出库练习 | 推动箱子会滚动并改变顶面 | 2-3 分钟 | 1/10 |
| 2 | 门铃测试 | 用箱子持续压按钮开门 | 3 分钟 | 2/10 |
| 3 | 清道夫巡线 | 冲击面可击败普通敌人 | 4 分钟 | 3/10 |
| 4 | 重货通道 | 重压面可压重压按钮并击败重甲敌人 | 5 分钟 | 4/10 |
| 5 | 一箱三用 | 同一个箱子依次完成开门、攻击、供能 | 6-8 分钟 | 5/10 |

## Level 1: 出库练习

- **区域**: 星港装配区 A-01
- **类型**: Puzzle Tutorial
- **目标**: 让玩家理解“推动不是平移，而是翻滚”
- **状态**: Concept

### 核心教学

- 玩家推动箱子后，箱子会滚动 1 格
- 滚动后，顶面图标会变化
- 现阶段不要求玩家记住所有功能，只要求注意“它会变”

### 关卡布局

```text
#######
#S....#
#..B..#
#.....#
#....E#
#######

S = 玩家起点
B = 箱子
E = 出口
```

### Critical Path

1. 玩家看到前方一只箱子和一个远处出口。
2. 走到箱子旁边并第一次推动。
3. 箱子翻滚，顶面图标明显变化。
4. 玩家继续把箱子推离通道，自己走向出口。
5. 过关后用一句简短文字总结：`箱子每滚一次，功能面都会变化。`

### 为什么需要这一关

如果玩家第一关就要记按钮和敌人，会忽略“滚动 = 状态变化”这个核心概念。第一关必须纯净。

---

## Level 2: 门铃测试

- **区域**: 星港装配区 A-02
- **类型**: Puzzle Tutorial
- **目标**: 教玩家用箱子压住普通按钮，让门保持开启
- **状态**: Concept

### 新教学点

- 箱子可以当重物使用
- 普通按钮需要“持续压住”
- 玩家要学会“先布置，再通过”

### 关卡布局

```text
########
#S..B..#
#..P.D##
#......#
#....E.#
########

P = 普通按钮
D = 关闭的门
```

### Critical Path

1. 玩家先尝试直接去门口，发现门是关的。
2. 玩家把箱子推到普通按钮上，门打开。
3. 玩家绕路从开启后的门通过。
4. 玩家抵达出口。

### 设计备注

- 这个关卡不强调箱子是哪一面，只强调“箱子可以持续占位”。
- 按钮和门之间的连线要非常直观。

---

## Level 3: 清道夫巡线

- **区域**: 星港清扫轨道 B-01
- **类型**: Puzzle + Light Combat
- **目标**: 教玩家用冲击面撞击普通敌人
- **状态**: Concept

### 新教学点

- 普通敌人会阻挡通路
- 不是所有顶面都能打败敌人
- 玩家需要先把箱子滚成正确面，再撞过去

### 关卡布局

```text
#########
#S.B....#
#...R...#
#...N.E.#
#########

R = 建议停留点，方便把箱子调整成冲击面
N = 普通敌人
```

### Critical Path

1. 玩家第一次把箱子直接推向敌人，若顶面不对则失败，得到明确提示。
2. 玩家观察顶面变化后，学会在走廊里先多滚一格或换方向。
3. 当冲击面朝上时，玩家把箱子撞向普通敌人。
4. 敌人被击败，通路打开。
5. 玩家抵达出口。

### 设计备注

- 敌人应长得像坏掉的小清洁机器人，不做威胁感过强的造型。
- 这里要首次强调“错误的推动会失败，但失败原因可理解”。

---

## Level 4: 重货通道

- **区域**: 重载升降井 B-03
- **类型**: Puzzle + Light Combat
- **目标**: 教玩家理解重压面的双用途
- **状态**: Concept

### 新教学点

- 重压按钮只能由重压面触发
- 重甲敌人只能被重压面击败
- 稀有功能面比冲击面更珍贵，要规划顺序

### 关卡布局

```text
##########
#S..B..H##
#....W..##
#..P..D.E#
##########

H = 重甲敌人
W = 调整路线的缓冲区
P = 重压按钮
D = 门
```

### Critical Path

1. 玩家发现普通推法无法处理重甲敌人。
2. 玩家在缓冲区调整箱子朝向，使重压面朝上。
3. 先击败重甲敌人或先压重压按钮，两种顺序都可测试。
4. 玩家意识到重压面是稀有状态，不应浪费。
5. 通过开启后的门抵达出口。

### 设计备注

- 这关开始可以允许一个很短的“错误恢复空间”，让玩家不必每次都重开整关。
- 但 MVP 仍以整关快速重置为主。

## Level 5: 一箱三用

- **区域**: 发货总控室 C-01
- **类型**: Mixed Puzzle
- **目标**: 让玩家首次完整体验“一个箱子承担三种职责”
- **状态**: Concept

### 新教学点

- 同一个箱子必须被重复利用
- 顺序错误会导致自己把路堵死
- 能源面引入“最终供能”这一目标

### 关卡布局

```text
############
#S.B..P....#
#....D..N..#
#......T...#
#......E...#
############

P = 普通按钮
D = 门
N = 普通敌人
T = 能源槽
E = 出口
```

### 推荐解法结构

1. 先让箱子压住普通按钮，打开第一道门。
2. 把箱子从按钮上取回，并重新规划滚动路径。
3. 让箱子以冲击面撞击普通敌人，清出通道。
4. 再次回收箱子，调整到能源面朝上。
5. 把箱子送入能源槽，点亮出口终端。
6. 玩家穿过最终通道并抵达出口。

### 这一关必须验证的体验

- 玩家是否第一次明确感受到“这个箱子真的是工具箱”
- 玩家是否在不看外部说明的情况下理解“先开门再回收再供能”
- 玩家是否愿意在失败后再尝试一种不同顺序

## Cross-Level Teaching Notes

### UI 提示节奏

- 第 1 关: 只提示“箱子会滚动并改变顶面”
- 第 2 关: 提示“按钮需要持续压住”
- 第 3 关: 提示“当前顶面无法击败该敌人”
- 第 4 关: 提示“重压面可压重载机关”
- 第 5 关: 不再显式教规则，只给目标提示

### 玩家常见误区

| 误区 | 最早出现关卡 | 解决方式 |
| ---- | ---- | ---- |
| 以为箱子像传统推箱子一样只是平移 | 1 | 放大翻滚动画和顶面图标变化 |
| 以为按钮按一次就永久开启 | 2 | 让门在移开箱子后立刻关闭 |
| 以为攻击是玩家技能，不是箱子功能 | 3 | 让玩家无法直接碰撞消灭敌人 |
| 以为重压面只用于机关，不用于战斗 | 4 | 同关中同时出现重压按钮和重甲敌人 |
| 以为箱子用完就结束，不需要回收 | 5 | 让关卡明确要求回收同一箱子 |

### MVP Completion Standard

若测试玩家在完成第 5 关后能主动说出类似下面的话，则教学弧线有效：

- “我要先把箱子滚成对的面再推过去。”
- “这个箱子先压门，之后还要拿回来打怪。”
- “我在想怎么让它最后变成能源面。”

如果玩家仍只把箱子理解为普通推箱道具，说明教学失败，应重做前 3 关的反馈与布局。
